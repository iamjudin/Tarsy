# Changelog

All notable changes to Tarsy are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project uses semantic versioning with a Codex cachebuster build suffix during local iteration.

## [Unreleased]

### Added

- Semantic auto-activation rules for conversational thinking, brainstorming, critique, option comparison, and tradeoff analysis prompts.
- Compatibility guardrails so Tarsy steps back for primary functional plugins, structured artifacts, formal documents, and high-stakes topics.
- GitHub-ready README with animated and static branding assets.
- Repository validation script for plugin metadata, public assets, and pet spritesheet checks.
- GitHub Actions CI workflow.
- Issue and pull request templates.
- Contribution guide.
- Pet install/update troubleshooting documentation.

### Changed

- Replaced placeholder branding with the prepared Tarsy public banner and icon assets.

## [0.1.0] - 2026-07-14

### Added

- Single dry sarcastic tone skill for Codex.
- Tone-only behavior that preserves honesty, accuracy, safety, and engineering judgment as non-adjustable behavior.
- Optional Tarsy Codex pet with a v2 animated spritesheet.
- Pet install script for copying the pet into `~/.codex/pets/tarsy`.

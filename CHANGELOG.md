# Changelog

All notable changes to Tarsy are documented here.

## 0.1.0 — 2026-07-14

First public release candidate.

- Add a single dry sarcastic tone skill for Codex.
- Preserve honesty, accuracy, safety, and engineering judgment as non-adjustable behavior.
- Package the optional Tarsy Codex pet with a v2 animated spritesheet.
- Add a pet install script for copying the pet into `~/.codex/pets/tarsy`.
